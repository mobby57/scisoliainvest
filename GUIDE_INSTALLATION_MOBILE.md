# Installation de l'Application SCI Solia Invest sur Votre Téléphone

## Vue d'ensemble

L'application SCI Solia Invest est maintenant disponible en tant qu'application web progressive (PWA). Cela signifie que vous pouvez l'installer directement sur votre téléphone depuis votre navigateur web, sans passer par l'App Store ou Google Play.

## Avantages de l'Installation

- ✅ Accès rapide depuis l'écran d'accueil de votre téléphone
- ✅ Fonctionne hors ligne (une fois chargée)
- ✅ Expérience similaire à une application native
- ✅ Mises à jour automatiques
- ✅ Notifications push (à venir)
- ✅ Pas besoin de télécharger depuis les stores d'applications

## Instructions d'Installation

### Sur Android (Chrome)

1. **Ouvrez le site** : Naviguez vers `https://votre-domaine.com` dans Google Chrome
2. **Installez l'application** :
   - **Méthode 1** : Cliquez sur le bouton "Installer l'app" en haut de la page
   - **Méthode 2** : 
     - Appuyez sur le menu (trois points verticaux ⋮) en haut à droite
     - Sélectionnez "Ajouter à l'écran d'accueil" ou "Installer l'application"
3. **Confirmez** : Appuyez sur "Installer" ou "Ajouter"
4. **Terminé** : L'icône de SCI Solia Invest apparaîtra sur votre écran d'accueil !

### Sur iPhone/iPad (Safari)

1. **Ouvrez le site** : Naviguez vers `https://votre-domaine.com` dans Safari
2. **Ouvrez le menu de partage** : Appuyez sur le bouton "Partager" (carré avec une flèche vers le haut) en bas de l'écran
3. **Ajoutez à l'écran d'accueil** : 
   - Faites défiler vers le bas
   - Appuyez sur "Sur l'écran d'accueil"
4. **Nommez l'application** : Le nom "SCI Solia Invest" sera suggéré
5. **Ajoutez** : Appuyez sur "Ajouter" en haut à droite
6. **Terminé** : L'application apparaîtra sur votre écran d'accueil !

### Sur Windows/Mac (Chrome, Edge, Safari)

1. **Ouvrez le site** dans votre navigateur
2. **Installez l'application** :
   - **Chrome/Edge** : Cliquez sur l'icône d'installation (⊕) dans la barre d'adresse
   - **Safari (Mac)** : Fichier > Ajouter au Dock
3. **Confirmez l'installation**
4. **Lancez depuis** votre barre des tâches, Dock, ou menu Démarrer

## Vérification de l'Installation

Pour vérifier que l'application est correctement installée :

1. Recherchez l'icône "SCI Solia Invest" sur votre écran d'accueil
2. L'icône devrait être bleue avec un "S" blanc
3. Ouvrez l'application - elle devrait s'ouvrir en plein écran sans les barres du navigateur

## Désinstallation

### Android
1. Maintenez l'icône de l'application enfoncée
2. Sélectionnez "Désinstaller" ou glissez vers "Supprimer"

### iPhone/iPad
1. Maintenez l'icône de l'application enfoncée
2. Appuyez sur "Supprimer l'app" ou le symbole "-"
3. Confirmez la suppression

### Windows/Mac
1. Faites un clic droit sur l'icône de l'application
2. Sélectionnez "Désinstaller" ou "Supprimer du Dock"

## Problèmes Courants

### Le bouton "Installer l'app" n'apparaît pas
- **Vérifiez votre navigateur** : Assurez-vous d'utiliser Chrome (Android) ou Safari (iPhone)
- **Connexion HTTPS** : L'application doit être servie via HTTPS
- **Déjà installée** : L'application est peut-être déjà installée

### L'application ne fonctionne pas hors ligne
- Ouvrez l'application au moins une fois avec une connexion Internet
- Le contenu sera mis en cache pour un accès hors ligne ultérieur

### L'icône ne s'affiche pas correctement
- Désinstallez et réinstallez l'application
- Videz le cache de votre navigateur
