# SCI Solia Invest - Backend

Simple backend server for SCI Solia Invest.

## Getting Started

### Installation

```bash
npm install
```

### Configuration

Copy `.env.example` to `.env` and update the values:

```bash
cp .env.example .env
```

### Development

```bash
npm run dev
```

### Production

```bash
npm start
```

## API Endpoints

- `GET /health` - Health check endpoint
- `GET /api/*` - API routes
