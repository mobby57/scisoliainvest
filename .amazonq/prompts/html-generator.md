# Générateur HTML - SCI Solia Invest

Créez du HTML moderne pour la plateforme d'investissement SCI.

## Template de base
```html
<!doctype html>
<html lang="fr">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="[DESCRIPTION]" />
  <meta name="theme-color" content="#1f2937" />
  <link rel="icon" href="/favicon.ico" />
  <title>[TITRE] - SCI Solia Invest</title>
</head>
<body>
  [CONTENU]
</body>
</html>
```

## Classes CSS
- `.modern-card` : Cartes avec hover
- `.btn-primary` : Boutons bleus
- `.progress-bar` : Barres orange
- `.animate-fade-in-up` : Animation

## Couleurs
- Bleu : `bg-blue-800`, `text-blue-900`
- Orange : `bg-orange-600`, `text-orange-500`
- Gris : `bg-gray-50`, `text-gray-900`

Utilisez Tailwind CSS et respectez la charte SCI.
